# ItemHitokoto

将一条「一言」命名到你喜欢的物品上吧！  
[鸣谢](#鸣谢)

## 使用

插件只会修改物品显示名称，其他 NBT 都不会修改，大家可以放心使用

### 指令

- `/itemhitokoto [句子类型]`
  - `句子类型` - _可选参数_  
    可用值：  
    `动画`（`a`）  
    `漫画`（`b`）  
    `游戏`（`c`）  
    `文学`（`d`）  
    `原创`（`e`）  
    `来自网络`（`f`）  
    `其他`（`g`）  
    `影视`（`h`）  
    `诗词`（`i`）  
    `网易云`（`j`）  
    `哲学`（`k`）  
    `抖机灵`（`l`）
