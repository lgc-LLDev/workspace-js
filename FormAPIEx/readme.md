# FormAPIEx

让处理表单返回值变得更容易！

## 介绍

本工具库提供了一些类以及函数，可以让各位更方便地处理表单返回值，不用再为需要自己处理表单按钮的 ID 而烦恼了！  
它甚至可以直接让你不需要写太多代码即可发送一个可以翻页以及搜索的列表！  
快来使用吧~

本工具库几乎全部使用异步语法编写，详细内容可以直接看下方的示例代码
